What Is This?

This a help document which explains all options for the Normal Map Generation Plugin. This document and all the related material is part of the Curvature Simulation Gem.

Launching The Plugin Dialog

The plugin is a Max4 Tool and you can find it in the Tool Tab, otherwise you can place it there by clicking in the "Configure Button Set" icon.

The Dialog

Low Object Button
Picks the low detailed object from any max viewport. If the high model is alredy picked, the process will start.
High Object Button
Picks the detailed object from any max viewport. If the low model is alredy picked, the process will start.
On Selected Faces
If pressed, the process only will use the selected faces of the low detailed mesh.
Non Continous Normal
The plugin will not share normals for the vertices with two or more texture coordinates (this can produce discontinuities).
In Texture Space
If pressed the output map will be in texture space (or tangent space) rather than in object space.
Map Size
Specifies the size of the output map.

Status Dialog

After both models are picked, the process will start. While the map is being generated an Status Dialog will show the progress the operation. The Low Priority button changes the thread priority to decrease the usage of CPU by the plugin's thread. The Cancel button will stop the progress, but will save all the generated map until the button was pressed (so if the map is big, canceling the operation can take some seconds)

Output

The generated map will be place at c:\normal.tga You can change it inside the source code.
