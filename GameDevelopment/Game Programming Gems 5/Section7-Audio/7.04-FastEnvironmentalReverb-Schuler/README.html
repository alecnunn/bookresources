Reverb Demo

written by Christian Schüler, 2004

synopsis

This program lets you play with various parameters of a 4-channel FDN in real time. A graph shows the theoretical transfer function of the FDN as a whole.

usage

Click on the ellipsis "..." next to the "input file" field and select a wave file in ".wav" pcm format or a file containing raw pcm data. The play button starts the action. The graph will start to animate if the delay lengths are modulated.

The task manager will show some ~10% CPU usage during playback. This is due to drawing the graph with windows GDI, it's not the cause of the reverb procedure. When you minimize the window, CPU usage should drop to below measureable.

Playback frequency and volume can be changed during playback.

If you press ENTER inside an editbox, the program will close. Press TAB instead.

output

The live sound output is also recorded to disk as a file named record.wav into the current directory, if such a file can be opened (may be locked by CoolEdit, for instance). Usually the current directory is set to the directory from where you opened the input file.

presets

Presets are contained in a .txt file with the name "presets.txt" which is parsed when the program starts. You can edit this file, see there for explanation.
  • "Temple of the ancestors" - Very long reverberation time, large room.
  • "Temple of the ancestors (back door)" - Same as above but with 100% occlusion of direct sound.
  • "Open Stage in the Woods" - Very large, echoey room.
  • "Cardio Fitness at the Gym" - Medium size room, bright spectrum.
  • "U-Boat Club Party" - Small room, strong coloration, high frequency damping
  • "Christians Former very small Dorm Room" - very small room
  • "Howling Spirits" - Short delay lines as oscillators, sweeping modulation gives pitch shift.
  • "Leaking Biohazard Container" - Short delay lines as oscillators, inharmonic resonant modes.
  • "Oversize Shoe Box" - Feedback matrix simulates sign change at open ended wave reflection, gives hollow sound.

parameters

Changing any parameter will have an immediate effect.

Feedback matrix
Delay lengths
Half lives
These work as described in the article.
Input and output rotation
The input and output matrices contain some sin/cos dependent on these parameters. This serves to somehow distribute the signal from stereo to 4-channel and back to stereo.
Delay length modulation
The delay lengths are modulated by a sinusoid of given period and depth.
Mix
Dry mix is the percentage of direct signal, wet mix is the percentage of reverberated signal in the final result.

others

Mute Input
Pretend the input to the reverb was silence. Good for listening to reverb responses.
Bypass
Listen to the original signal instead of the reverberated one. The reverb keeps updating in the background.
Reset
Zero all delay lines.
