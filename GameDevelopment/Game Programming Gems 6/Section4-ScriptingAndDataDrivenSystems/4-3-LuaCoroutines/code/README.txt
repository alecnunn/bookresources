Programming Advanced Control Mechanisms with Lua Coroutines  
-----------------------------------------------------------

by Luiz Henrique de Figueiredo, Waldemar Celes, Roberto Ierusalimschy

e-mail: team@lua.org



The distributed code illustrates different control mechanisms built on top of Lua coroutines.

The content of the directory is:

    + filter.lua: use of Lua coroutine to implement a filter.
    + iterator.lua: use of Lua coroutine to implement an iterator.
    + scheduler.lua: use of Lua coroutine to implement a simple task scheduler.
    + dispatcher.lua: use of Lua coroutine to implement a cooperative multitasking.
    + lua.exe: Lua stand-alone interpreter for Windows (using Lua 5.1a).

To download the latest version of Lua, visit www.lua.org.
